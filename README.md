# 📄 Resume Dataset Analysis Project

## 📌 Overview
This project focuses on analyzing and preprocessing a resume dataset to extract meaningful insights. The dataset contains multiple resumes that can be used for tasks like classification, skill extraction, and candidate profiling.

## 📊 Dataset
- File Name: `resume_dataset_200k_enhanced.csv`
- Type: Structured CSV data
- Contains: Resume text, skills, job roles, and other candidate-related information

## 🎯 Objective
- Clean and preprocess resume data
- Perform exploratory data analysis (EDA)
- Extract useful features (skills, experience, etc.)
- Prepare data for machine learning models

## 🛠️ Tools & Technologies
- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## 🔍 Steps Performed
1. Data Loading
2. Data Cleaning (handling missing values, duplicates)
3. Text Processing (if applicable)
4. Exploratory Data Analysis (EDA)
5. Visualization of key insights

## 📈 Possible Use Cases
- Resume classification
- Job recommendation systems
- Skill extraction using NLP
- Candidate ranking systems
